## border-radius:值  	设置长方形的圆角	 值的取值如下
```
                x1 x2 x3 x4 / y1 y2 y3 y4  	
                左上角  长半轴是x1  短半轴是y1的椭圆角	
                右上角  长半轴是x2  短半轴是y2的椭圆角
                右下角  长半轴是x3  短半轴是y3的椭圆角
                左下角  长半轴是x4  短半轴是y4的椭圆角
八个值 上左 上右 下右 下左 椭圆角	x1 x2 x3 x4 / y1 y2 y3 y4
六个值 上左 上右 下左 下右 椭圆角	x1 x24 x3 / y1 y24 y3
四个值	左上右下 右上左下 椭圆角 x13 x24 / y13 y24
四个值	左上 右上 右下 左下 圆角 x1 x2  x3 x4
三个值	左上 右上左下 右下
两个值	左上右下 右上左下
一个值      所有方位角的圆角都是一样的

单位  px   %
```

## vertical-align:值				设置该元素与父元素行框盒子的垂直对齐方式
```
            baseline  				基线对齐 //默认
            middle  				中线对齐
            top/bottom				line-height的顶部/底部	
            text-top/text-bottom	 文本的顶部/底部
            px		  	 			举个栗子 1px   	表示相对于基线向上移动1px     上正下负
            %						相对于line-height的百分比
```
