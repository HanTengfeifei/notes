```
关于标签
什么是标签		标签是HTML(超文本标记语言)的最小组成单位  
标签的组成		标签是由一对尖括号组成的  例如<img>
标签的作用		盛放网页中文字和图片等信息
```

## 制作页面中常用的标签
```
div标签				网页中最常用的标签	
	<div>我是一个div标签</div>
p标签				段落标签
	<p>如果你想在网页中插入一个段落,那就用我吧</p>
hx 					标题标签
	<h1>我是网页的一级标题</h1>	h1标签在整张网页有且只能出现一次
	<h2>我是网页的二级标题</h2>
span标签			span标签和p标签一块使用
	<p>我是<span>海文</span>老师</p>
img标签				图片标签
	<img  src=‘1.jpg’ width='' height='' alt=''>	如果你想在网页中插入一张图片 那就使用图片标签吧
a标签				超链接标签
	<a href=‘www.baidu.com’>	如果你想跳转到百度 那就点我吧</a>	
	
列表标签
	ul 	li    			无序列表标签组
		ul  li 是一组标签			在网页中表示一组没有顺序的列表
		消除默认小原点：list-style-type:none;
        <ul>
            <li></li>
            <li></li>
        </ul>
	ol 	li 				有序列表标签组
		ol  li  是一组标签			在网页中表示一组有顺序的列表
        <ol>
            <li></li>
            <li></li>
        </ol>
	dl	dt  dd				自定义列表标签
		dl  dt  dd 是一组标签		在网页中表示一组你自己定义的列表
        <dl>
            <dt>列表标题</dt>
            <dd>列表内容</dd>
            <dd>列表内容</dd>
        </dl>
        
b标签&&strong标签
	b标签			是规定粗体文本的标签
	strong标签		不仅可以规定粗体文本       而且还能够表示强调  给搜索引擎看

i标签&&em标签	
	i标签			是规定斜体文本的标签
	em标签			不仅可以规定斜体文本       而且还能够表示强调 给搜索引擎看

sub标签&&sup标签
	sub标签			是规定下标的标签
	sup标签			是规定上标的标签
```

## 标签的分类

```
双标签	<div></div>       双标签是由一个开始标签和结束标签组成的
单标签	<img>	         单标签只有一个开始标签
```
